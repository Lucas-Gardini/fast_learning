FastLearning #02 - Centralizando com css



Baixar Arquivos

Neste vídeo foi mostrado como centralizar elementos do HTML com css, os códigos css respectivos estão abaixo: 

                
/* Utilizar somente para páginas sem scroll! Recomendando para elementos de tamanho médio e estáticos! */
/* Centralizando utilizando posições absolutas */

.centralizar01 {
position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%)
}

/* Utilizar em elementos-pai, para centralizar seus elementos. Geralmente o mais utilizado, por ter um posicionamento relativo */

.centralizar02 {
position: relative;
width: 100%;
height: 100%;
display: grid;
place-items: center;
}

/* Centralizando textos (somente horizontalmente) */

.centralizar03 {
text-align: center;
}

/* Utilizando o mesmo método da primeira classe, porém centralizando somente verticalmente */

.centralizar04 {
position: absolute;
top: 50%;
transform: translateY(-50%);
}
